# Model Training and Updates

This project provides a simple model for training and updating a local coding model.

## Usage

1. Create a `Model` instance and train it with a list of `CodingTask` instances.
2. Fine-tune the model for a specific domain using the `fine_tune` method.
3. Update the model with new tasks using the `update` method.
4. Save the model to a file using the `save` method.
5. Load the model from a file using the `load` method.
